# Interfaces and Implementations

## Interface Prefix

- Use `I` prefix for interface classes.
- Examples: `IApiMagDecDts`, `IExportRepo`, `IRepository`.

## Implementation Suffix

- Use `Impl` suffix for concrete implementations.
- Examples: `MagDecRepoImpl`, `NoaaApiMagDecDtsImpl`.
- Pattern: implement interface, append `Impl` to base name.
