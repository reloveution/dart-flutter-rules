# Priority Guidelines

## Dart/Flutter Packages

1. **Dart MCP** pub_dev_search — ALWAYS first
2. Firecrawl scrape — for additional context from pub.dev pages

## Documentation

1. Dart MCP pub_dev_search — for package documentation links
2. Firecrawl scrape — for web docs not via other tools

## Web Search

1. Firecrawl map + scrape — targeted scraping (search unavailable)
2. web_search — general internet search

## Code Analysis

1. Dart MCP — all Dart/Flutter analysis

## Project Work

1. Dart MCP — all development tasks

## ML/AI Tasks

1. Hugging Face MCP — models, datasets, papers, HF docs, image gen
2. Firecrawl — ML content not on HF Hub
