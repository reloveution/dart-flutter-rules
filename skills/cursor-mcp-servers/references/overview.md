# General Principles

## Before Use

- Always verify tool availability before use.
- Use the most appropriate server for the specific task.

## Server Purposes

- **Dart MCP:** Dart/Flutter development tools (analysis, tests, pub.dev).
- **Firecrawl:** Web scraping, internet information (instead of web_search).
- **Hugging Face:** ML models, datasets, research papers, image generation.

## Right Tool

- Don't use web search when a specialized MCP tool exists.
- Dart packages → Dart MCP pub_dev_search.
- ML models/datasets → Hugging Face MCP.
- Web content → Firecrawl scrape/map (search currently unavailable).
