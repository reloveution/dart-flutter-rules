# PR-Level Checks

22. **Scope**: Change set is focused and scoped to the stated purpose; no unrelated or unnecessary changes.
23. **Description**: PR/MR description accurately reflects the changes made.
24. **Tests**: New or updated tests cover new or changed logic.
25. **Test quality**: For each test, evaluate whether it could fail or only checks mock implementation.
26. **CI**: All tests pass in the continuous integration system.
