# Extensions and Mixins

## Extensions

- Use `X` suffix for extension names.
- If multiple extensions exist, organize them in `X` folder.
- Example: `ExtExcelPrjBaseSheet` in `X/` folder.

## Mixins

- Use `Mxn` suffix for mixins.
- Examples: `MxnExcelFormatting`, `MxnRepoCharsXlsxDisc`.
- Distinguishes mixins from classes at a glance.
