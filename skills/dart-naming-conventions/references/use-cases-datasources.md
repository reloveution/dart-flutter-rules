# Use Cases and Datasources

## Use Cases

- Use `Ucs` suffix for use cases.
- Examples: `GetMagDecUcs`, `ExecuteProjectExportUcs`.
- Clear identification of use case classes.

## Datasources

- Use `Dts` suffix for datasources.
- Interface: `IApiMagDecDts`, `IExcellExportDts`.
- Use `repo` instead of `repository` in class and folder names.
- Use `dts` instead of `datasource` in folder and variable names.
