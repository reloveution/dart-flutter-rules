# General Naming Rules

## File Naming

- Use `snake_case` for file names.
- Examples: `user_service.dart`, `api_client.dart`.

## Class Naming

- Use `PascalCase` for classes.
- Examples: `UserService`, `ApiClient`.

## Member, Variable, Function

- Use `camelCase` for members, variables, functions, and enums.
- Examples: `userName`, `fetchUserData()`.

## Acronyms

- Capitalize acronyms and abbreviations longer than two letters as words.
- **Good:** `HttpClient`, `SqlRepository`.
- **Bad:** `HTTPClient` (all caps for multi-char acronym).
