# Data Structures and Algorithms

## Efficient Structures

- Use efficient data structures for performance-critical operations.
- Choose the right structure: List vs Set vs Map; growable vs fixed.
- Prefer O(1) or O(log n) over O(n) when scale matters.

## Algorithms

- Use efficient algorithms for performance-critical operations.
- Avoid nested loops when linear or better alternatives exist.
- Cache expensive computations when inputs don't change.
