# Profiling

## Flutter DevTools

- Profile app performance regularly using Flutter DevTools.
- Identify bottlenecks: build time, layout, paint, rasterize.
- Use Timeline, CPU profiler, memory view.

## When to Profile

- Before and after major optimizations.
- When investigating jank or slow startup.
- Periodically during development to catch regressions.
