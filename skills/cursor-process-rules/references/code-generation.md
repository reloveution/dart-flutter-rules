# Code Generation

## Dynamic vs Static

- Use actual dynamic code generators rather than hard-coded static code examples.
- Generate real, runnable code suited to the context.
- Avoid placeholder or fake implementations.

## File Names and Thoroughness

- Always specify the exact file name when writing code.
- Be thorough and attentive; don't rush.
- Provide complete, ready-to-use code for each step.
